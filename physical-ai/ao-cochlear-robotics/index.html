Overview

AO develops soft robotic tools for cochlear implant surgery — a procedure requiring insertion of a delicate electrode array into the inner ear's spiral canal. Conventional rigid robots risk basilar membrane trauma. This project built the full ML system: from patient-specific simulation data generation through cloud deployment and continuous model auto-update, ensuring the surgical robot operates within safe parameters.

System Architecture

200+ Patient Cochlear Models (CT scans → 3D geometry) │ ▼ SOFA Physics Simulation (FEM soft actuator behavior) 300 physical parameters × AWS HPC distributed sweep │ ▼ 100X+ speedup vs. sequential Synthetic Dataset (force profiles · insertion trajectories) │ ▼ ML Model Training (PyTorch) ├── Force → safe insertion depth regression ├── Safety classifier (trauma threshold detection) └── Uncertainty quantification (MC Dropout) │ ▼ Cloud Deployment (AWS) + Auto-Update Pipeline └── Drift detection → retrain trigger → evaluation → deploy

Safety by Design

Every model version passes an automated evaluation suite before deployment — measuring false negative rate on unsafe trajectory detection. The system uses MC Dropout uncertainty quantification to flag low-confidence predictions for human review, ensuring no borderline case reaches the surgical robot without validation.
