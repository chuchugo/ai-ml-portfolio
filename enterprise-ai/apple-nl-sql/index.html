Overview

Apple's global marketing teams needed real-time campaign performance insights — but accessing the data required SQL expertise and slowed decision-making. This system lets any marketer ask questions in plain English and get back validated, deterministic ROAS and ROI numbers in under 4 seconds, integrated directly with Apple's internal data warehouse.

Architecture

Natural Language Query (marketer) │ ▼ Intent Parser — entity extraction (metrics · dimensions · filters · timeframes) │ ▼ Schema RAG — retrieve relevant tables/columns from vector index of Apple data warehouse schema │ ▼ SQL Generator (LLM, temp=0) Schema-grounded query construction │ ▼ Validation Pipeline ├── Syntax check ├── Semantic consistency (query matches intent?) ├── Row-count safeguard (no accidental full scans) └── PII field blocklist enforcement │ ▼ Execution → Apple Internal Data Warehouse │ ▼ ROAS · ROI · CTR · CPC — formatted for business users
