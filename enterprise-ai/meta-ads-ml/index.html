Overview

Meta's advertising platform serves 7M+ advertisers across billions of daily impressions. Effective ad targeting requires real-time behavioral signals processed at petabyte scale and ML models that translate those signals into precise user segments and conversion probability scores. This work was done in direct collaboration with Meta's Ads ML team on model architecture and feature iteration.

System Architecture

Behavioral Event Stream (petabyte/day) ├── Clicks · views · purchases · engagement └── Sampled at sub-second granularity │ ▼ Feature Pipeline (Kafka → Spark → Feature Store) ├── Real-time features: session signals, recency ├── Batch features: LTV scores, content affinity └── Cross-feature interactions + embeddings │ ┌────┴─────┐ │ │ User Ad Targeting Segmentation Classifiers (clustering) (CTR + CVR) │ │ └────┬─────┘ │ Ad Auction Scoring (7M+ advertisers · billions of impressions/day)
