Overview

Engineering teams at Alibaba were firefighting — discovering problems only after user-facing incidents. Existing tooling was siloed: logs in one system, metrics in another, traces nowhere. This project unified all three into a single observability pane of glass built on Kubernetes, Elasticsearch, and a custom APM stack — shifting the organization from mean-time-to-detect (MTTD) of hours to minutes.
